# ds-svelte

ds-svelte contains [Aksel](https://aksel.nav.no) components in Svelte.

## Installation

```bash
# Login with a GitHub Personal Access Token
npm login --scope=@nais --auth-type=legacy --registry=https://npm.pkg.github.com

# Add @nais to .npmrc
echo "@nais:registry=https://npm.pkg.github.com" >> .npmrc

# Add ds-svelte to your project
npm install @nais/ds-svelte --save-dev
```
